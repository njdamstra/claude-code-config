The problem: What if multiple unsubscribes happen simultaneously?

- Line 588: Gets count = 2
- Line 589: Sets count = 1
- Line 590: Checks 1 === 0? No, continues

- Line 589: Sets count = 1 (overwrites Thread 1's value!)

const count = (eventCounts.get(event) || 0) - 1
