### The shadcn-vue-mcp Server

**Created by HelloGGX**, this MCP server provides AI-driven component generation specifically for the shadcn-vue component library. It integrates seamlessly with Claude Code, Cursor, VSCode (with Copilot or Cline), and other MCP-compatible IDEs.[2][1]

### Core Features

The shadcn-vue MCP server provides several powerful tools:[3][1]

1. **`component-builder`** - AI-powered Vue component generation with shadcn-vue and Tailwind CSS integration
2. **`components-filter`** - Smart component recommendation system that analyzes your requirements
3. **`component-usage-doc`** - Real-time access to shadcn-vue documentation with interactive previews
4. **`component-quality-check`** - Automated A11y compliance checking and code quality assessment
5. **`requirement-structuring`** - Analyzes natural language requirements and converts them into structured formats

### How It Works

Unlike the official shadcn MCP server (which connects to registries), the shadcn-vue MCP takes a different approach by generating components through AI based on natural language descriptions. You simply describe what you need, and it generates production-ready Vue 3 components that comply with shadcn-vue and Tailwind CSS standards.[1][2][3]

Example usage:
```
/ui create a navigation bar component
/refine optimize the navbar's responsiveness and accessibility
/check
```

### Installation Options

The easiest way to get started is through **Smithery** (a hosted MCP service):[1]

1. Visit the shadcn-vue-mcp page on Smithery
2. Click "Generate URL" to get your personal connection URL with API key
3. Add it to your IDE's MCP configuration file

**For Claude Code** (`.mcp.json`):
```json
{
  "servers": {
    "shadcn-vue-mcp": {
      "url": "https://server.smithery.ai/@HelloGGX/shadcn-vue-mcp/mcp?api_key=xxxx&profile=..."
    }
  }
}
```

**For Cursor** (`~/.cursor/mcp.json`):
```json
{
  "servers": {
    "shadcn-vue-mcp": {

  }
}
```
