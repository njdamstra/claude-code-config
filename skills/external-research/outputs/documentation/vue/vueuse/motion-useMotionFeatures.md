# useMotionFeatures

useMotionFeatures is used to register features such as variant sync, event listeners.

## Parameters

### `instance`

A motion instance from [useMotion](https://motion.vueuse.org/api/use-motion) or [useMotions](https://motion.vueuse.org/api/use-motions).

### `options`

The same options object as [useMotion](https://motion.vueuse.org/api/use-motion).
