# Research Outputs

This directory stores cached documentation research findings.

## File Naming Conventions

- **Official docs:** `doc_[topic]_[YYYY-MM-DD].md`
- **Web research:** `web_[topic]_[YYYY-MM-DD].md`

## Cache Lifetime

Files older than 60 days are considered stale and may be replaced with fresh research.

## Organization

Files are automatically organized by date in filename. No subdirectories needed.
