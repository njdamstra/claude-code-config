# What are integrations?

Integrations are a way to connect your application to external services and enhance your Groq-powered applications with additional capabilities.
Browse the categories below to find integrations that suit your needs.

[AI Agent Frameworks](https://console.groq.com/docs/integrations#ai-agent-frameworks) [Browser Automation](https://console.groq.com/docs/integrations#browser-automation) [LLM App Development](https://console.groq.com/docs/integrations#llm-app-development) [Observability and Monitoring](https://console.groq.com/docs/integrations#observability-and-monitoring) [LLM Code Execution and Sandboxing](https://console.groq.com/docs/integrations#llm-code-execution-and-sandboxing)

[UI and UX](https://console.groq.com/docs/integrations#ui-and-ux) [Tool Management](https://console.groq.com/docs/integrations#tool-management) [Real-time Voice](https://console.groq.com/docs/integrations#realtime-voice)

## [AI Agent Frameworks](https://console.groq.com/docs/integrations\#ai-agent-frameworks)

Create autonomous AI agents that can perform complex tasks, reason, and collaborate effectively using Groq's fast inference capabilities.

[![Agno](https://console.groq.com/integrations/agno_black.svg)\\
\\
Agno\\
\\
Agno is a lightweight library for building Agents with memory, knowledge, tools and reasoning.](https://console.groq.com/docs/agno) [![AutoGen](https://console.groq.com/integrations/autogen.svg)\\
\\
AutoGen\\
\\
AutoGen is a framework for building conversational AI systems that can operate autonomously or collaborate with humans and other agents.](https://console.groq.com/docs/autogen) [![CrewAI](https://console.groq.com/integrations/crewai.png)\\
\\
CrewAI\\
\\
CrewAI is a framework for orchestrating role-playing AI agents that work together to accomplish complex tasks.](https://console.groq.com/docs/crewai) [![xRx](https://console.groq.com/integrations/xrx.png)\\
\\
xRx\\
\\
xRx is a reactive AI agent framework for building reliable and observable LLM agents with real-time feedback.](https://console.groq.com/docs/xrx)

## [Browser Automation](https://console.groq.com/docs/integrations\#browser-automation)

Automate browser interactions and perform complex tasks and transform any browser-based task in to an API endpoint instantly with models via Groq.

[![Anchor Browser](https://console.groq.com/integrations/anchorbrowser.png)\\
\\
Anchor Browser\\
\\
Anchor Browser is a browser automation platform that allows you to automate workflows for web applications that lack APIs or have limited API coverage.](https://console.groq.com/docs/anchorbrowser)

\\
LlamaIndex\\
\\

\\
LiteLLM\\
\\
LiteLLM is a library that standardizes LLM API calls and provides robust tracking, fallbacks, and observability for LLM applications.](https://console.groq.com/docs/litellm) [![Vercel AI SDK](https://console.groq.com/vercel-integration.png)\\

Vercel AI SDK\\
\\

[![Composio](https://console.groq.com/integrations/composio_black.png)\\
\\
Composio\\
\\

Toolhouse\\
