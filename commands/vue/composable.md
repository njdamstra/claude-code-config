# Vue Composable Builder

Invoke the `vue-composable-builder` skill to design and build Vue 3 composables with proper architecture, lifecycle management, store integration, and SSR safety.

**Skill invocation:** Use the Skill tool with parameter `vue-composable-builder`

The skill will handle:
- Composable architecture patterns
- Lifecycle management (onMounted, onUnmounted, watchEffect)
- Nanostore integration for reactive state
- SSR safety with client-side guards
- VueUse integration patterns
- Type-safe return values with TypeScript
