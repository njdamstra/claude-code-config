# VueUse Library Expert

Invoke the `vueuse-expert` skill to become skilled in using VueUse's impressive library of composables.

**Skill invocation:** Use the Skill tool with parameter `vueuse-expert`

The skill will handle:
- VueUse composable selection and usage
- SSR-safe patterns with VueUse
- Common use cases (useLocalStorage, useMouse, useEventListener, etc.)
- Advanced patterns (useAsyncState, useRefHistory, etc.)
- Integration with Vue 3 Composition API
- Performance optimization with VueUse utilities
