# Vue Component Builder

Invoke the `vue-component-builder` skill to build Vue 3 components using Composition API, script setup syntax, TypeScript strict mode, and Tailwind CSS.

**Skill invocation:** Use the Skill tool with parameter `vue-component-builder`

The skill will handle:
- Vue 3 Composition API patterns
- TypeScript strict mode compliance
- Tailwind CSS styling (no scoped styles)
- Dark mode support with `dark:` utilities
- SSR safety with `useMounted()` checks
- Form validation with Zod schemas
- Accessibility with ARIA labels
