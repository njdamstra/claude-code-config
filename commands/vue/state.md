# Vue State Management Architecture

Invoke the `vue-architect` skill to design state management architecture for Vue 3 applications by deciding where state should live and preventing common anti-patterns.

**Skill invocation:** Use the Skill tool with parameter `vue-architect`

The skill will handle:
- State placement decisions (local vs shared vs global)
- Nanostore architecture patterns
- Prop drilling prevention
- Reactive state composition
- SSR-safe state initialization
- Anti-pattern detection and prevention
