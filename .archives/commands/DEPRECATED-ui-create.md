# ⚠️ DEPRECATED COMMAND

This command has been **replaced by `/ui-new`** with enhanced features.

## Migration

```bash
# Old
/ui-create Badge with status colors

# New
/ui-new Badge with status colors
```

## What Changed

The new `/ui-new` command includes:
- ✅ **Planning phase with ASCII wireframes** (skip with `--skip-plan`)
- ✅ **Flexible variant generation** (`--variants N`)
- ✅ **Optional validation** (`--skip-validation`)
- ✅ **Optional screenshot** (`--screenshot`)

## See Also
- `/ui-new` - Full component creation workflow
- `/ui-plan` - Planning only (with wireframes)
- `/ui-review` - Quality review after creation
